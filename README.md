# Enterprise-IT-Support-Service-Desk-Operations-Homelab
This repository documents hands-on IT infrastructure and systems administration work completed within a self-built Windows Server 2022 home lab environment. Each section covers a distinct operational area of enterprise IT — from identity and access management through to endpoint deployment, remote support, and data protection — and is documented to professional standards with configuration details, process rationale, and where applicable, supporting screenshots.
The work here is designed to demonstrate practical competency relevant to IT Support Technician, Desktop Support, Help Desk, and Junior Systems Administrator roles. All configurations reflect standard enterprise practices.
# 🖥️Environment
Windows Server 2022 (Domain Controller)  |  Windows 10 / Windows 11 (Domain-joined clients)  |  Active Directory Domain Services | spiceworks
